# 好书推荐 Web 应用

一个基于 Vue.js 和 Node.js 的书籍推荐网站，用户可以浏览、搜索、推荐和收藏书籍。

## 功能特性

- 📚 浏览书籍列表
- 🔍 搜索书籍和作者
- 🏷️ 按类型筛选书籍
- 👍 推荐喜欢的书籍
- ❤️ 收藏心仪书籍
- ➕ 添加新书籍

## 技术栈

- **前端**: Vue.js, HTML5, CSS3
- **后端**: Node.js, Express.js
- **数据库**: MongoDB Atlas
- **部署**: Heroku

## 安装和运行


```bash
npm install
npm run seed
npm start  or npm run dev
